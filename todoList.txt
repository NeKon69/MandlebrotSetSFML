Производительность и стабильность:
Dynamic Thread Block Size: Автоматический подбор размера блока потоков.
Умный CPU Fallback: Автоматическое определение доступности CUDA, сравнение производительности GPU/CPU.
Ограничение времени выполнения CUDA-ядра: "Безопасная песочница" для защиты от зависаний.


Новые возможности рендеринга:
Orbit Trap Rendering: Создание необычных визуальных эффектов.
Distance Estimation Method (DEM): Более точная отрисовка, сглаживание, 3D-эффекты.
3D Rendering: Базовый 3D-рендер фрактала с освещением и тенями (на основе DEM).
Экспорт изображения высокого разрешения: 16k, 32k для печати.
Мультисемплирование: Реализация антиальясинга через мультисемплирование.
Адаптивное сглаживание: Автоматическая регулировка уровня сглаживания в зависимости от детализации изображения и производительности GPU.


Для "кодеров фракталов":
Custom Expression Parser: Парсер математических выражений в рантайме (комплексные числа, функции, переменные).
Шаблоны CUDA-ядер: Шаблоны с комментариями для быстрого старта.
Подсветка синтаксиса и отладка CUDA-кода в IDE: (Плагин/расширение для VS Code, Visual Studio).
Возможность изменения формул Julia Set в рантайме


Палитры:
Визуальный редактор палитры: Добавление, удаление, изменение цветов.
Градиентные палитры: Несколько цветовых точек.
Импорт/Экспорт палитр: .gpl и другие форматы.
Случайные палитры: Генерация случайных палитр.
